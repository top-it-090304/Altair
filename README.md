**🎮Platformer Adventure** — 2D платформер для ОС «Аврора»

![Platformer Adventure](images/icon.png)
---
**📖 О проекте**

**Platformer Adventure** — это двухмерный платформер, разработанный на Godot Engine для операционной системы «Аврора».
Игроку предстоит пройти серию из 32 уровней, преодолевая препятствия, избегая ловушек и демонстрируя точность управления. Главная цель — дойти до конца уровня.
Проект выполнен в ярком 2D стиле.

---
**✨ Ключевые особенности**

 - Новые механики по мере прохождения
