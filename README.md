## 📖 О проекте

**Приключение Платформера** — это двухмерный платформер, разработанный на **Godot Engine** для операционной системы **«Аврора»**.

Игроку предстоит пройти серию из **24 уровней**, преодолевая препятствия, избегая ловушек и демонстрируя точность управления. Главная цель — дойти до конца уровня и собрать все фрукты.

Проект выполнен в ярком, красочном 2D-стиле.

## 🧗 Платформенные механики

- **Базовый прыжок** — стандартное перемещение по уровню
- **Двойной прыжок** — дополнительный прыжок в воздухе
- **Исчезающие платформы** — платформы, которые пропадают под ногами
- **Ловушки** — препятствия, требующие точности
- **Чекпоинты** — точки сохранения прогресса

## 💻 Управление на ПК (для тестов)

| Клавиша | Действие |
|:---:|:---|
| `W` `A` `S` `D` | Движение |
| `Пробел` | Прыжок |
| `Пробел × 2` | Двойной прыжок |
| `ESC` | Пауза |
