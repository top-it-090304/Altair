## 📖 О проекте

**Приключение Платформера** — это двухмерный платформер, разработанный на **Godot Engine** для операционной системы **«Аврора»**.

Игроку предстоит пройти серию из **24 уровней**, преодолевая препятствия, избегая ловушек и демонстрируя точность управления. Главная цель — дойти до конца уровня. Проект выполнен в ярком 2D-стиле.

- 🗺️ 24 проработанных уровня

| **Пробел** | Прыжок |
| **Двойной пробел** | Двойной прыжок |
